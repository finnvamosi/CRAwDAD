# CRAwDAD: Causal Reasoning Augmentation with Dual-Agent Debate

</div>

This repository contains code associated with the paper:
> Vamosi, F. G., & Forkert, N. D. (2025) CRAwDAD: Causal reasoning augmentation with dual-agent debate. arXiv. https://arxiv.org/abs/2511.22854

## Usage

Clone the repository and install dependencies:

```bash
git clone https://github.com/finnvamosi/CRAwDAD.git
cd CRAwDAD
pip install -r requirements.txt
```

Run code:

```bash
python run.py [arguments]
```

Please see `run.py` for all arguments.
